# Credit Card Fraud Dataset

This directory should contain the creditcard.csv file.

## Download Instructions

1. Download the dataset from: https://www.kaggle.com/mlg-ulb/creditcardfraud
2. Place the `creditcard.csv` file in this directory
3. The file will be automatically tracked with Git LFS due to its large size (143.84 MB)

## Note

The actual CSV file is not included in the repository due to its size.
It will be tracked with Git LFS when you add it.
